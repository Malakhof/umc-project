# Система квалификации лидов — УМЦ Калуга

## 1. Скоринговая система (1-10 баллов)

### Критерии оценки

| Критерий | 1-3 балла (низкий) | 4-6 баллов (средний) | 7-10 баллов (высокий) |
|----------|-------------------|---------------------|----------------------|
| **Размер компании** | ИП, до 5 сотрудников | 6-50 сотрудников | 50+ сотрудников |
| **Срочность** | "Когда-нибудь потом" | "В ближайшие 1-2 месяца" | "Нужно срочно / проверка ГИТ / предписание" |
| **Бюджет** | Не выделен, ищут бесплатное | Есть понимание стоимости, ограничен | Бюджет утвержден, готовы платить |
| **ЛПР (лицо, принимающее решение)** | Рядовой сотрудник "просто узнать" | Специалист ОТ / HR-менеджер | Директор / зам. директора / главный инженер |

### Расчет итогового балла

```
Итоговый балл = (Размер + Срочность × 1.5 + Бюджет × 1.3 + ЛПР × 1.2) / 5

Максимум: 10 баллов
```

**Весовые коэффициенты:**
- Срочность — x1.5 (самый важный фактор, горящие сроки = быстрая сделка)
- Бюджет — x1.3 (готовность платить сокращает цикл)
- ЛПР — x1.2 (контакт с ЛПР ускоряет решение)
- Размер компании — x1.0 (базовый)

### Категории лидов по баллам

| Балл | Категория | Цвет в таблице | Действие |
|------|-----------|----------------|----------|
| 8-10 | **HOT** (горячий) | Красный | Звонить в течение 1 часа. Приоритет №1 |
| 5-7 | **WARM** (теплый) | Желтый | Связаться в течение 4 часов. Подготовить КП |
| 3-4 | **COOL** (прохладный) | Синий | Связаться в течение 24 часов. Прогрев |
| 1-2 | **COLD** (холодный) | Серый | Добавить в рассылку. Контакт раз в 2 недели |

## 2. Стадии воронки продаж

```
НОВЫЙ → КОНТАКТ → КВАЛИФИКАЦИЯ → КП ОТПРАВЛЕНО → ПЕРЕГОВОРЫ → СДЕЛКА / ОТКАЗ
```

### Детальное описание стадий

| Стадия | Описание | Максимальное время в стадии | Действие менеджера |
|--------|----------|---------------------------|-------------------|
| **НОВЫЙ** | Заявка поступила с сайта / звонок / email | 1 час (HOT), 4 часа (WARM), 24 часа (COOL) | Первичный звонок |
| **КОНТАКТ** | Первый разговор состоялся | 2 рабочих дня | Квалификация, выявление потребности |
| **КВАЛИФИКАЦИЯ** | Потребность выявлена, параметры определены | 3 рабочих дня | Подготовка и отправка КП |
| **КП ОТПРАВЛЕНО** | Коммерческое предложение отправлено | 5 рабочих дней | Контрольный звонок через 2 дня |
| **ПЕРЕГОВОРЫ** | Обсуждение условий, согласование | 10 рабочих дней | Работа с возражениями, корректировка условий |
| **СДЕЛКА** | Договор подписан, оплата получена | — | Передача в работу, планирование обучения |
| **ОТКАЗ** | Клиент отказался | — | Зафиксировать причину, поставить на прогрев через 3 месяца |

### Правила перехода между стадиями

- Если лид "застрял" на стадии дольше максимального времени — автоматический сигнал менеджеру
- После 3 безответных попыток связи — перевод в "ОТЛОЖЕН" (возврат через 2 недели)
- Причина отказа обязательно фиксируется (для аналитики)

## 3. Квалификационные вопросы для первого звонка

### Блок 1: Выявление потребности (обязательно)

1. **"Какие программы обучения вас интересуют?"**
   - Руководители и специалисты (охрана труда)
   - Специалист по охране труда
   - Уполномоченный по охране труда
   - Пожарная безопасность
   - Комплексное корпоративное обучение
   - Консультация

2. **"Сколько сотрудников нужно обучить?"**
   - 1-5 человек → малая группа
   - 6-15 человек → средняя группа (возможна скидка)
   - 16-30 человек → большая группа (скидка 10-15%)
   - 30+ человек → корпоративный формат

3. **"В какие сроки нужно провести обучение?"**
   - Срочно (предписание ГИТ, авария, проверка) → приоритет максимальный
   - В ближайший месяц → стандартный приоритет
   - В перспективе / планируем → прогрев

### Блок 2: Бюджет и принятие решения

4. **"Есть ли у вас выделенный бюджет на обучение по охране труда?"**
   - Да, утвержден → уточнить сумму
   - Нет, но руководство одобрит → узнать процедуру согласования
   - Нет, пока только изучаем → прогрев

5. **"Кто принимает окончательное решение о выборе учебного центра?"**
   - Я сам(а) → ЛПР, работаем напрямую
   - Руководитель / директор → нужен выход на ЛПР
   - Тендерная комиссия / закупки → долгий цикл, уточнить сроки

6. **"Вы уже рассматриваете другие учебные центры?"**
   - Нет, только вас → хорошо, не затягиваем
   - Да, сравниваем → уточнить критерии, подготовить сравнительные аргументы
   - Да, у нас есть текущий провайдер → работа с возражением "уже есть поставщик"

### Блок 3: Дополнительные потребности

7. **"Нужна ли помощь с оформлением документации по ОТ?"**
   - Допродажа консультационных услуг

8. **"Как часто вы проводите обучение? Есть ли регулярная потребность?"**
   - Разовое → стандартная работа
   - Регулярное (раз в квартал / полгода) → долгосрочный контракт, VIP-условия

9. **"Удобен ли выездной формат обучения на вашей территории?"**
   - Для групп 10+ человек — предложить выездное обучение

## 4. Матрица решений

### Когда ПРИОРИТИЗИРОВАТЬ (работать в первую очередь)

| Сигнал | Действие |
|--------|----------|
| Предписание ГИТ / плановая проверка через 1-2 месяца | Звонить немедленно, предложить ускоренный формат |
| Группа 15+ человек | Подготовить КП со скидкой в тот же день |
| ЛПР на связи, бюджет есть | Назначить встречу / видеозвонок в течение суток |
| Произошел несчастный случай на производстве | Максимальный приоритет, предложить срочное обучение + консультацию |
| Повторный клиент просит новую программу | VIP-обслуживание, персональные условия |

### Когда ПРОГРЕВАТЬ (поддерживать контакт)

| Сигнал | Действие |
|--------|----------|
| "Нам пока не срочно, планируем на следующий квартал" | Поставить напоминание, отправить полезные материалы раз в 2 недели |
| Нет бюджета, но потребность есть | Отправить обоснование для руководства (ROI, штрафы ГИТ) |
| Контактное лицо — не ЛПР | Попросить контакт ЛПР, параллельно прогревать текущий контакт |
| Малая группа (1-3 чел.), стоимость до 15 000₽ | Автоматизировать: отправить ссылку на запись, шаблонное КП |

### Когда ДИСКВАЛИФИЦИРОВАТЬ (не тратить время)

| Сигнал | Действие |
|--------|----------|
| Ищут бесплатное обучение / "у вас есть бесплатные курсы?" | Вежливо объяснить стоимость, если не готовы — закрыть лид |
| Не из Калужской области и не готовы к дистанционному формату | Порекомендовать центр в их регионе |
| Не могут назвать ни одного конкретного параметра (кто, сколько, когда) | Отправить информацию на email, поставить на прогрев |
| Запрос не по нашему профилю (промбезопасность, экология, медосмотры) | Вежливый отказ с рекомендацией |
| 3+ касания без ответа, нет реакции ни на что | Перевести в "спящие", возврат через 3 месяца |

## Быстрая шпаргалка для менеджера

```
ПОЛУЧИЛ ЗАЯВКУ →
  1. Открой карточку лида, заполни основные поля
  2. Оцени по скорингу (4 критерия)
  3. HOT (8-10)? → Звони СЕЙЧАС
     WARM (5-7)? → Звони в течение 4 часов
     COOL (3-4)? → Звони сегодня
     COLD (1-2)? → Email + прогрев
  4. Задай квалификационные вопросы (блоки 1-3)
  5. Заполни карточку полностью после разговора
  6. Перемести в следующую стадию
  7. Поставь дату следующего контакта
```
